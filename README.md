# Sparky Mouse
Sparky Mouse a free, safe, open source asset for your website. It creates a image element on the webpage and it follows your cursor anywhere in the document without blocking other element's event listeners. It's developed for custom cursors for your website.
# Installation
To install first open ``mouse.js`` and download the raw file, or just open releases and select the latest release. After downloaded upload ``mouse.js`` to the root of your site. After it's uploaded your website, go into any webpage and import the script.
```html
<script src="mouse.js"></script>
```
If the file you're imported into is not located in the root simply change the script element's source path.
